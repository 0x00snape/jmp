# jmp
JMP is a simple rust tool that makes easy to navigate to a directory in the file system. Use this tool on my personal space if you want, use it...

## Quick-Install
```bash
git clone
cd jmp
cargo build --release 
```
<b> Getting rust binary </b>
```bash
  jmp() {
	  cd "$(~/jmp/target/release/jmp $1)"  
  }
```
